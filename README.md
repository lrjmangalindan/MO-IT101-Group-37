---

### Program Details
This project is a **console-based Java Payroll System** written in a **single `.java` file** without using Object-Oriented Programming.

The system:
- Loads employee data from `resources/employees.csv`
- Processes **5,000+ attendance records** from `resources/attendance.csv`
- Supports login for **employees** and **payroll staff**

Payroll staff can calculate employee work hours, apply government deductions, and generate the **final net salary per employee**.
